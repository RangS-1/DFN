# DFN - Design Feedback Normalizer

![DFN Logo](icon.png)

**DFN** is a simple tool to **normalize and structure messy design feedback** collected from various sources (clients, reviewers, team members, surveys, etc.) into a clean, consistent, and easy-to-read format.

**Currently in very early development stage.**

## Planned / Current Features

- [ ] Read raw feedback from manual input
- [ ] Clean and standardize formatting (bullet points, categories, priorities)
- [ ] Export results to txt

## Installation

1. Clone the repository

```bash
git clone https://github.com/RangS-1/DFN.git
cd DFN
```
